📖 관리자 사용 방법

이 페이지에서 시장 홈페이지를 직접 관리할 수 있습니다
📣 공지사항 관리
  • ✅ + 공지 작성 버튼을 누르면 새 공지를 올릴 수 있어요
  • ✅ 유형을 이벤트로 선택하면 행사 안내가 됩니다
  • ✅ 상단 고정으로 설정하면 맨 위에 표시됩니다
  • ✅ 숨김을 누르면 방문자에게 안 보입니다
🏪 상인 관리
  • ✅ + 상인 등록으로 시장 상인을 추가할 수 있어요
  • ✅ 이모지(🥩🌿🍄)를 넣으면 이미지처럼 활용됩니다
  • ✅ 등록하면 홈페이지 상인 소개 칸에 자동으로 나타납니다
📋 예약 관리
  • ✅ 방문자가 신청한 예약이 대기중으로 들어옵니다
  • ✅ 확정 버튼으로 예약을 승인하세요
  • ✅ 위쪽 버튼으로 대기중 / 확정 / 취소를 나눠서 볼 수 있어요
🪟 팝업 관리
  • ✅ + 새 팝업 만들기로 팝업을 여러 개 만들 수 있어요
  • ✅ 이미지 칸을 클릭하면 사진을 올릴 수 있습니다
  • ✅ 활성화 버튼을 누른 팝업 1개만 홈페이지에 표시됩니다
  • ✅ 팝업을 끄려면 비활성화 버튼을 누르세요
⚠️ 중요: 데이터는 이 컴퓨터 브라우저에 저장됩니다. 다른 기기에서 관리하려면 같은 브라우저를 사용하세요.
